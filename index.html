    lume-theme-docs

    A theme for documentation using Lume.

    Install

    Set up Lume and add the following to your _config.ts:

    import lume from "lume/mod.ts";
import docs from "https://raw.githubusercontent.com/ansanloms/lume-theme-docs/main/mod.ts";

const site = lume();

site.use(docs());

export default site;

    This theme uses JSX Preact for its layout, so add the following to the compilerOptions in your deno.json:

    {
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "npm:preact"
  }
}
